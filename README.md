# Customer Behavior Analysis — Data Analytics Project 

A complete end-to-end Data Analyst project where I analyzed customer shopping behavior using Python (Jupyter Notebook), PostgreSQL, and Power BI.
This project covers data cleaning, exploratory data analysis, SQL business queries, and a Power BI dashboard for insights-driven decision-making.

# Project Overview 

This project aims to understand how customers shop, what influences their purchase decisions, and how different factors like gender, subscriptions, discounts, shipping type, and age groups affect revenue.

I performed:

Data Cleaning & Preprocessing – using Python

Exploratory Data Analysis (EDA) – using Python

Business Insights & SQL Queries – using PostgreSQL

Interactive Visualization Dashboard – using Power BI

# Key Objectives

Analyze customer purchasing trends

Identify top products and revenue drivers

Compare spending behaviors of different customer groups

Understand impact of discounts, shipping type, and subscriptions

Build an interactive dashboard for business decision-making
